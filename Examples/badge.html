    Badge

    Badges are used to highlight status or something about the object for recognition.

    We provide you with 2 standard badges : Error and Success

    Error
    Success

    Badge on Avatar

    These badges can be used on our avatars to provide online and offline status.

    Badge on Icon

    This badge can be used on icons to show any small update or number on top-right of icons.

    2

    Usage

    A div with class badge-container must be the parent for all badges.
    Inside the div, you can use another div class badge error or badge success for colored badges.

    For avatar badges, inside the container div, use a div below the avatar with class badge-avatar.
    Aditionally you can also use classes online offline for the color of avatar badge.

    Note: Use the class largeand class xlarge for avatars that are large and xlarge.

    For icon badges inside the container div, use a div below the icon img with class badge-cion.
    Aditionally you can use classes offline online to get colored badges.
